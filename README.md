# 🌐 Link in Bio - Pedro Ivo

Uma página de links personalizada (estilo Linktree) construída para centralizar meu portfólio, redes sociais e contatos profissionais. Desenvolvida com foco em **Mobile First**, garantindo uma experiência fluida e rápida para usuários vindos do Instagram.

🔗 **[Acesse o projeto online aqui](https://pedao-cod.github.io/link-in-bio-PS/)** *(ou coloque seu link encurtado aqui)*

---

## 📱 Demonstração

O layout foi desenhado utilizando a tendência **Glassmorphism** (efeito de vidro translúcido) sobre um fundo escuro (Dark Mode) com animações dinâmicas de partículas, passando uma identidade tecnológica e moderna.

## ✨ Funcionalidades

- **Design Responsivo:** Adaptado perfeitamente para telas de celulares e desktops.
- **Efeito Glassmorphism:** Interface translúcida e elegante usando `backdrop-filter`.
- **Fundo Interativo:** Animação de partículas interativas com o mouse/toque usando a biblioteca `particles.js`.
- **WhatsApp API:** Botões de contato direto para o WhatsApp com mensagens pré-configuradas (URL Encoded) para facilitar o orçamento de sistemas fotovoltaicos.
- **Hover Effects:** Feedback visual suave ao interagir com os botões.

## 🚀 Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando as seguintes tecnologias:

- **HTML5:** Estruturação semântica.
- **CSS3:** Estilização avançada com Flexbox, variáveis nativas (`:root`) e efeitos visuais.
- **JavaScript:** Configuração e inicialização da biblioteca de animação de fundo.
- **[Particles.js](https://vincentgarreau.com/particles.js/):** Biblioteca leve para a criação das partículas no canvas.
